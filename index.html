Logistic Regression Practice (1D)

Paste the CSV dataset (x,y) and parameter updates (b,a) to visualize and animate the sigmoid fit.

1) Dataset (x,y) CSV

Format: two columns x,y with header optional.

2) Parameter updates (b,a) CSV

Paste your results here. Each row is an update: b,a (intercept first, slope second; header optional).

3) Plot

What’s happening?

  • Blue points are class 0 plotted at y=0; red points are class 1 at y=1.
  • The grey dashed line is y = 0.5.
  • We animate sigmoid(x; a, b) = 1 / (1 + exp(-(a·x + b))) using your (b,a) updates (intercept first, slope second).
Front-end only. No frameworks. Uses Chart.js for plotting.
